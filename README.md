# Stipendio

Stipendio is a web application designed to help users calculate and track their work hours and earnings. Built with React, it provides an intuitive interface for entering hourly rates, logging work hours, and viewing summaries. The app integrates with Firebase for authentication and data storage, ensuring a seamless and secure user experience.

## Features

- **Work Hours Input:** Easily log daily work hours.
- **Hourly Rate Management:** Set and update your hourly wage.
- **Summary Tables:** View detailed and total summaries of your work and earnings.
- **Authentication:** Secure login and user management via Firebase.
- **Responsive UI:** Modern, responsive design for desktop and mobile.
- **Data Persistence:** All data is stored securely in Firestore.
- **Dialogs & Confirmations:** User-friendly dialogs for editing, deleting, and confirming actions.
- **Testing:** Comprehensive unit tests for hooks, utilities, and database logic.
